# Personal Finance Tracker
A Java-based console application designed to manage daily financial transactions using Object-Oriented Programming principles.

## Features
- **Transaction Tracking:** Add and categorize income and expenses.
- **Dynamic Calculation:** Real-time balance updates using polymorphic logic.
- **Reporting:** Generate summary reports with category-wise expense breakdowns.
- **OOP Implementation:** Utilizes Inheritance, Abstraction, Encapsulation, and Polymorphism.

## How to Run
1. Download the `Main.java` file.
2. Open your terminal or Command Prompt.
3. Compile the file: `javac Main.java`
4. Run the program: `java Main`
